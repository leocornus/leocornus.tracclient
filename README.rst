WordPress Trac Client Plugin
============================

**wp-trac-client** is a WordPress_ plugin trying to provide
a complete solution to seamlessly integrate Trac_ project 
into WordPress_.
This plugin is using XML-RPC APIs to talk to Trac_ instance.
