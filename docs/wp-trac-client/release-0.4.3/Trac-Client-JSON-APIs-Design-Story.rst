Requirements
------------

We need a set of APIs for AngularJS_ apps to communicate with
Trac projects through wp-trac-client plugin.

Basic actions:

- query
- create ticket
- update ticket
- delete ticket

Should we create a AngularJS_ app called **angular-trac-client**.
